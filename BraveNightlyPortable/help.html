Brave Nightly Portable

Unofficial PortableApps.com Format package for Brave Browser Nightly.

Launch

From the repository root, double-click START-BRAVE-NIGHTLY.bat.

Or use Launch Brave Nightly.bat in this folder.

Updates

Automatic on every launch via the wrapper exe. Source: official GitHub Nightly releases only.
