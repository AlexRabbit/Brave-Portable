Brave browser binaries are not stored in this repository.

On first launch, START-BRAVE-NIGHTLY.bat (or the wrapper exe) downloads the
official Nightly build from:

  https://github.com/brave/brave-browser/releases

Only releases tagged "Nightly" with brave-v*-win32-x64.zip are used.
ProductName must be "Brave Browser Nightly" before installation.
